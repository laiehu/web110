HTML Elements

Element Description Block, Inline, or N/A Container or Empty
<!--...--> Used to insert comments in the source code to help keep track of and take note of changes and work. N/A Container
<!DOCTYPE> An instruction to the web browser about what version of HTML the page is using. N/A Empty
<a> Used to anchor a link to another site or file to the page. Inline Container
<article> Specifies independent, self-contained content. The element should be able to stand alone. Inline Container
<aside> tag defines some content aside from the content it is placed in. The page content should be related to the <aside> content. Block Container
<body> Used to define the Pages's body. Block Container
<br /> Inserts a single line break. Inline Empty
<dd> used to describe a term defined by the <dl> tag. Inline Container
<div> Defines a general division or section on a page. Block Container
<dl> used to define a discription list determined by the <dl> tag. Inline Container
<em> used to render contained text as emphasized. Inline Container
<field> surrounds related content in a box, grouping them. Inline Container
<figcaption> defines caption for <figure> element. Inline Container
<figure> marks-up specified content. Inline Container
<footer> creates container for information such as authorship, functionality checks, or related documents. Block Container
<form> creates HTML form for user interaction. Inline Container
<h1>-<h6> marks-up text into specifically stylized heading categories. Inline Container
<head> contains head elements such as scripts and styles. Block Container
<header> contains header elements such as <h1> elements, logos, and icons. Block Container
<html> contains all other html elements. Block Container
<iframe> specifies embedded inline document frame. Inline Container
<img /> defines an image in an html page. Inline Container
<input> defines field for user input. Inline Container
<label> defines label for button, meter, input, or other various fields for things like read assists. Inline Container
<li> used to define the elements of a list, which is either ordered <ol> or unordered <ul>. Inline Container
<link /> used to define a link from a document to an external resource or style sheet. Inline Container
<img /> used to display an image. Inline Container
<nav> used to define a set of navigation links. Block Container
<ol> used to define a set of an ordered list. Inline Container
<p> used to define a set of text in paragraph form. Inline Container
<script> used to define a client-side script common in JavaScript for image manipulation or validation. Inline Container
<section> used to define a section of a document, similar to header, footer, or chapter. Inline Container
<strong> used to define important text that is emphasized. Inline Container
<style> used to define style information used to style the HTML document. Inline Container
<table> used to define an html table, consisting of rows, header, and cells as well as styling markups. Inline Container
<td> used to define a cell in an HTML table. Inline Container
<th> used to define a header cell in an HTML table. Inline Container
<title> used to define a title for an HTML document, commonly reflected in the browser toolbar or title handle. Inline Container
<tr> used to define a row in an HTML table. Inline Container
<ul> used to define an unordered list. Inline Container
